# Sales + Onboarding Playbook (First 100 Tanzania Companies)

Date: February 8, 2026
Audience: Sales reps, implementation coordinators, customer success

## 1) Ideal Customer Profile (ICP)
- Tanzania SMEs with 10-300 employees.
- Monthly payroll pain: manual spreadsheets, delayed filings, compliance risk.
- Sectors: hospitality, retail, logistics, professional services.

## 2) Discovery Questions (First Call)
1. How many employees are paid monthly?
2. How do you currently calculate PAYE/SDL and produce filings?
3. How many days does payroll take from attendance close to payout?
4. What was your biggest payroll/compliance issue in the last 6 months?
5. Who signs off payroll and who owns tax submissions?

## 3) Demo Flow (20 Minutes)
1. Show employee onboarding and CSV import.
2. Show leave/timesheet import and rejected-row feedback.
3. Run payroll draft and explain PAYE/SDL/BIK handling.
4. Generate filing pack and show amendment/status tracking.
5. Show reports page cards: deadlines, SLA monitor, retention health.

## 4) Conversion Offer
- Pilot promise: first live payroll in 7 days with assisted setup.
- Risk control: parallel run against legacy for first cycle.
- Trust anchor: backup + restore drill and audit trail visibility.

## 5) Suggested Pricing Model (Simple, Sticky)
- Platform base fee + per-active-employee monthly fee.
- Example structure:
  - Starter (10-40 employees): TZS 180,000 base + TZS 6,000 per employee
  - Growth (41-150): TZS 300,000 base + TZS 5,000 per employee
  - Scale (151-500): Custom quote + SLA package
- Add-ons: assisted filings, premium support, custom integrations.

## 6) Onboarding Handover Checklist (Sales -> Implementation)
- Signed order form
- Named payroll approver and backup approver
- Company legal details
- Employee master data file
- Current month attendance/timesheet source
- Current period payroll summary from legacy system

## 7) 7-Day Onboarding Flow
1. Day 1: Kickoff, data template handover
2. Day 2: Employee + contract import
3. Day 3: Leave/timesheet import validation
4. Day 4: Draft payroll run and variance review
5. Day 5: Filing pack review and signoff
6. Day 6: First live run prep
7. Day 7: Go-live and owner confirmation

## 8) Stickiness Motions (Reduce Churn)
- Weekly payroll health report to owners.
- Deadline reminders visible in-app.
- 24h response SLA during payroll week.
- Quarterly compliance review with action list.

## 9) Objection Handling (Short)
- "We already use spreadsheets": keep spreadsheets as import source and cut processing time.
- "Compliance is risky": filing workflow + reminders + audit logs reduce missed deadlines.
- "Implementation will be hard": assisted first cycle and parallel run are included.
